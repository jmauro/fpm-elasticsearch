[fpm-elasticsearch](https://github.com/stankevich/fpm-elasticsearch)
======

[fpm-cookery](https://github.com/bernd/fpm-cookery) recipe for building elasticsearch .deb package.

## Usage

	fpm-cook
	sudo dpkg -i pkg/*.deb
	fpm-cook clean
